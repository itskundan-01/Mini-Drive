📋 STEP-BY-STEP:

1️⃣  CREATE USER ACCOUNT
   → Go to: http://localhost:5173
   → Click "Sign Up"
   → Fill in: Name, Email (admin@test.com), Password (admin123)
   → Click "Sign Up"

2️⃣  PROMOTE TO ADMIN
   → Open terminal
   → Run: cd backend
   → Run: node promoteToAdmin.js admin@test.com
   → You'll see: "✅ Successfully promoted to admin!"

3️⃣  LOGIN AS ADMIN
   → Logout from current session (if logged in)
   → Login with: admin@test.com / admin123
   → You'll be redirected to /admin/dashboard automatically!
